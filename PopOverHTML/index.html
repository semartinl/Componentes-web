Popover content

Existen diferentes tipos de popovers:

  1. Modals: Los cuales no se pueden interactuar con la demás pagina mientras se encuentre el popover. La API POPOVER solamente proporciona popovers no-modals
  2. No-modals: Los cuales dejan de mostrar la información cuando se establece algún evento con la página sobre la que está el popover.
